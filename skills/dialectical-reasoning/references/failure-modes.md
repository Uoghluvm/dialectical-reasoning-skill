# Failure Modes

Use this file when the draft analysis sounds clever but is not yet dialectical.

## Static Binarism

Symptom:

- "Either planning or market must win."
- "Either efficiency or equality matters."

Correction:

- Rebuild the object as a moving system.
- Ask how the poles condition one another.
- Ask which contradiction is principal at this stage.

## False Balance

Symptom:

- "Both sides have pros and cons."

Correction:

- Name the principal contradiction.
- Name the principal aspect.
- State what materially changes if the leading aspect changes.

## Abstract Universalism

Symptom:

- "History always proves..."
- "Dialectics says everything turns into its opposite."

Correction:

- Reinsert the concrete conditions.
- Specify the mechanism, threshold, and timeframe.
- If you cannot name the mediation, the claim is too abstract.

## Moral Substitution

Symptom:

- Replacing structural analysis with praise or blame.

Correction:

- Separate normative evaluation from contradiction analysis.
- Ask what material incentives, institutions, or capacities reproduce the pattern.

## Level Confusion

Symptom:

- Mixing individual motives, firm behavior, state strategy, and civilizational trends as if they were one level.

Correction:

- Split the analysis by level.
- Show how contradictions differ across levels and where they connect.

## Frozen Principal Contradiction

Symptom:

- Treating today's principal contradiction as eternal.

Correction:

- Ask what evidence would show a stage change.
- State which secondary contradiction might become principal next.
